# mocker
> API mock server

## Setup

```js
# install deps
npm install

# build app
npm run build

# build & launch app
npm run start

# run unit test
npm run test
```

## API

+ `GET /templates` checkout current templates
+ `PUT /templates/{url}` create/update specify template
+ `DELETE /templates/{url}` remove specify template
+ `PUT /templates/upload` override current templates
